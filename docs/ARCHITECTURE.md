# Architecture

This document outlines the architecture of the Gisele project.

## Project Structure

The project is structured as a standard Python package:

- `gisele/`: The main Python package containing the core logic.
- `scripts/`: Contains executable scripts, such as `run_gisele.py`.
- `data/`: Contains input data for the model.
- `docs/`: Contains project documentation.
- `tests/`: Contains tests for the `gisele` package.
- `setup.py`: The setup script for the `gisele` package.
- `requirements.txt`: A list of Python dependencies.
- `environment.yml`: The Conda environment file.
